Знайди кращу ціну ;)

  • Слова що знаходяться в лапках є голосовими командами наприклад "Голосові команди"

  • 1)Скажіть назву бренду або його номер, наприклад, "Самсунг"

  • 2) Для пролистування сторінок "Вперед" або "Назад"

  • 3) Для прокрутки сторінки "Вгору" або "Вниз"

  • 4) Для входу "Увійти", та виконайте подальші дії

  • 5) Для створення аккаунту "Реєстрація", та виконайте подальші дії

  • 6) Для того, щоб увійти "Увійти", та виконайте подальші дії

  • 7)Скажіть назву телефону або його номер від "1" до "12"

  • 8) Для переходу в особистий кабінет "Кабінет"

  • 9) Для повернення на головну сторінку "Головна"

  • 10) Для того, щоб побачити більше магазинів з даним товаром "Магазини"

  • 11) Для пошуку "Пошук"

  • 12) Для сортування по ціні "Мінімальна ціна" або "Максимальна ціна", а потім суму яку бажаєте

  • 13) Для сортування від дорогих до дешевих "Дорогі"

  • 14) Для сортування від дешевих до дорогих "Дешеві"

  • 15) Для перелистування фотографій конкретного продукту наприклад "Третє фото"

  • 16) Для перегляду всих характеристик продукту "Характеристики"

  • 17) Для пролистування сторінок з магазинами "Вперед" або "Назад"
